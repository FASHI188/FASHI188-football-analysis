# FASHI188 Football Analysis Runtime

This private repository is dedicated exclusively to the football-analysis project.

It stores football runtime data, executable code, validation assets, frozen evidence,
manifests, model diagnostics and audit outputs under `football-data/`.

Formal football rules are not stored in this repository. The sole formal rule source
remains the unique file whose name contains `CURRENT_唯一正式规则` in the ChatGPT football project.

This repository was created from a clean football-only snapshot of the former HHH1 runtime.
No HHH1 Git history, stock-analysis code, investment-analysis code, quote-bus assets or
legacy stock workflows are intentionally carried into this repository.

Football-specific GitHub Actions workflows are migrated separately after the runtime baseline
because the default Actions token cannot create or update workflow files without workflows permission.
